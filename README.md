# Awarded!

#### Conceito de Rede Social onde todo usuário pode visualizar ou fazer posts em suas respectivas categorias.

_Visite:_ https://awarded-artur.herokuapp.com/

Features do usuário:
- Login;
- Registro;
- Redefinição de senha por E-mail;
- Perfil;
- Edição de perfil (foto, e-mail, nome);
- Postar, editar ou excluir o próprio post;

Features do blog:
- Visualizar posts por categoria (tags);
- Visualizar todos os posts de um usuário;
- Visualizar single posts;
- Paginação;
- Data de postagem;


Desenvolvimento:<br>
Frontend: HTML, CSS, Bootstrap <br>
Backend: Python@Django<br>
Storages: S3 Bucket AWS<br>
Host: Heroku<br>
